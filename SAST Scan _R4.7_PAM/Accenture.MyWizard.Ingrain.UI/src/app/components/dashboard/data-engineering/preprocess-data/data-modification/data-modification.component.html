Techniques to deal with Outliers:
  • The user has an option to replace the outliers in the identified input attributes with Mean, Median, Mode, CustomValue or None. In case the data distribution is continuous and symmetrical, i.e. normally distributed, we can replace outliers using Mean. If the dataset is skewed or is of ordinal type, then Median can be used. In case data is of nominal type, we can replace outliers using Mode. To replace outliers using some business logic, it is best to use the Custom Value Option
  • SMOTE - SMOTE is an oversampling technique that generates synthetic samples from the minority class in case of imbalanced categorical Target attribute.
  • Interpolation - Interpolation is the process of estimating unknown values that fall between the known values for sequential data. The technique is applicable only for time-series models.

Skewness: • There are three techniques available to address the issue of data skewness.
  • Box Cox – The Box-Cox procedure automatically identifies a transformation from the family of power transformations that are indexed by a parameter λ which varies from -5 to 5. All values of λ are considered and the optimal value for your data is selected. The “optimal value” is the one which results in the best approximation of a normal distribution curve.
  • Reciprocal – This technique is used for non-zero values only. The value will be replaced by it's inverse.
  • Log – Commonly used for reducing right skewness and is often appropriate for measured variables. It cannot be applied to zero or negative values.

Techniques to deal with Data Imbalance:
Binning – Binning is a data processing technique used to reduce the imbalance present in the identified attributes. In this technique, the dataset points which are in minority and thus creating imbalance in data are grouped into a bin, and the original data values are replaced by a value representative of that group OR Binning is a way to bucketize/group less or imbalanced data replaced by a value representative of that group
